В качестве тестового задания предлагаю написать на любом языке консольный калькулятор (можно с веб-интерфейсом), который принимает входную строку, содержащую математическое выражение (целые и десятично-дробные числа, знаки +, -, *, / и скобки) и выводит в консоль результат его вычисления. Главным критерием при оценке задания является использование кандидатом при разработке методологии TDD, принципов SOLID и KISS. Архитектура решения должна обеспечивать расширение списка поддерживаемых операций при минимальном и максимально безболезенном для существующей функциональности внесении изменений в исходный код. Код должен быть легко читаем и отформатирован в едином стиле, содержать минимальное число поясняющих комментариев.
Исходный код должен быть выложен на GitHub в виде публично доступного репозитория.

Пример консольного ввода:
Введите выражение: 1+2-3
Результат: 0
